MEET MY WORKS


CHIP8

This was a project I worked on with 6 people to develop an emulator that executed CHIP-8 games on Google Chrome within 4 months. We used HTML/CSS, JavaScript and GitHub during the project. I was responsible for the user interface and specifically worked on creating a development environment to make CHIP-8 games targeted towards new users.


ICON PROJECT

For this project, I designed an app icon for a drone coffee delivery app for my graphic design class (IAT 102). The icon was done using adobe illustrator. The goals of this project was to learn how to convey meaning through images while considering the target audience. We were not allowed to use any words, letters or colours in the icon.
